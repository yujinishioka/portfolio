# Portfólio — Vinicius Yuji Nishioka

Portfólio pessoal desenvolvido com Next.js, apresentando projetos, experiência profissional e informações de contato. Disponível em português e inglês.

## Tecnologias

- **Next.js 15** com App Router
- **TypeScript**
- **Tailwind CSS**
- **next-intl** para internacionalização (pt / en)

## Rodando localmente

```bash
npm install
npm run dev
```

Acesse [http://localhost:3000](http://localhost:3000).

## Estrutura de conteúdo

Todo o conteúdo textual do site está nos arquivos de mensagens:

- `messages/pt.json` — versão em português
- `messages/en.json` — versão em inglês

Para atualizar textos, projetos ou experiências, edite esses arquivos.

## Deploy

O projeto está hospedado na [Vercel](https://vercel.com). Qualquer push para a branch `main` dispara um novo deploy automaticamente.
